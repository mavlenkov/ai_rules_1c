# Commands Specification — Delta

## MODIFIED Requirements

### Requirement: Поддержка серверных информационных баз
Команды должны работать с серверными ИБ через `/S` с аутентификацией. Формат подключения — без пробела после флага, без кавычек.

#### Scenario: Загрузка конфигурации в серверную ИБ
- **WHEN** в `infobasesettings.md` указано подключение `/Sserver:port\basename` с `/N 'UserName'`
- **THEN** конфигурация загружается через сервер 1С:Предприятие
- **AND** строка подключения передаётся без пробела после `/S` и без внешних кавычек

#### Scenario: Выгрузка из серверной ИБ
- **WHEN** в `infobasesettings.md` указано подключение `/Sserver:port\basename`
- **THEN** конфигурация выгружается с сервера без пробела после `/S`

#### Scenario: Серверная ИБ в Mode 2 (Designer fallback)
- **WHEN** используется прямой вызов Designer для серверной ИБ
- **THEN** рекомендуется формат `/IBConnectionString 'Srvr="server";Ref="base";'` как явный и однозначный

#### Scenario: Невалидная строка подключения с пробелом
- **WHEN** строка подключения содержит пробел после `/S` (например `/S server\base`)
- **THEN** Designer завершается с exit code 0, но выдаёт «Неопределена информационная база»
- **AND** инструкция предупреждает: это ошибка подключения, не успех

### Requirement: Поддержка файловых информационных баз
Команды должны работать с файловыми ИБ через `/F`. Формат подключения — без пробела после флага.

#### Scenario: Загрузка конфигурации в файловую ИБ
- **WHEN** путь к файловой ИБ указан как `/F/path/to/InfoBase`
- **THEN** конфигурация загружается из XML-файлов в указанную базу
- **AND** строка подключения передаётся без пробела после `/F`

#### Scenario: Выгрузка из файловой ИБ
- **WHEN** путь к файловой ИБ указан как `/F/path/to/InfoBase`
- **THEN** выгружаются объекты конфигурации без пробела после `/F`

## ADDED Requirements

### Requirement: Диагностика ошибок развёртывания
Инструкции развёртывания ДОЛЖНЫ содержать секцию диагностики типичных проблем.

#### Scenario: Ошибка подключения к ИБ
- **WHEN** в логе Designer обнаружена строка «Неопределена информационная база»
- **THEN** инструкция указывает: это ошибка подключения (неверный формат `/S` или `/F`)

#### Scenario: Штатное предупреждение для расширений
- **WHEN** в логе Designer обнаружена строка «Загрузка не должна менять принадлежность»
- **THEN** инструкция указывает: это штатное предупреждение, не ошибка

#### Scenario: Логи конвертера недоступны
- **WHEN** после выполнения скрипта 1CFilesConverter нужны логи для диагностики
- **THEN** инструкция рекомендует `bash -x` для трейса команд, т.к. логи удаляются автоматически

#### Scenario: Exit code 0 не гарантирует успех
- **WHEN** Designer завершился с exit code 0
- **THEN** инструкция предупреждает: необходимо дополнительно проверить лог на ошибки
