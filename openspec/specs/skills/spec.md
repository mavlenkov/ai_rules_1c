# Skills Specification

## Purpose
Предоставить углублённые знания и пошаговые инструкции для выполнения специфических задач в 1С.

## Requirements

### Requirement: Специализированные навыки для 1С
Навыки должны покрывать работу с метаданными, формами, макетами, ролями и платформой.

#### Scenario: Создание управляемой формы
- GIVEN нужно создать новую форму для объекта метаданных
- WHEN агент использует навык `1c-metadata-manage` с подкомандой form-scaffold
- THEN создаются файлы метаданных XML, Form.xml и Module.bsl с корректной структурой

#### Scenario: Компиляция макета из JSON DSL
- GIVEN есть описание табличного документа в JSON
- WHEN агент использует подкоманду mxl-compile
- THEN генерируется Template.xml с правильными именованными областями

### Requirement: Иерархическая структура навыков
Сложные навыки могут содержать подкоманды для детальных операций.

#### Scenario: 1c-metadata-manage как мета-навык
- GIVEN нужна операция с метаданными
- WHEN агент открывает `1c-metadata-manage/SKILL.md`
- THEN находит список всех подкоманд: form-*, mxl-*, role-*, epf-*, meta-*, db-*, cf-*, cfe-*

### Requirement: Утилитарные навыки
Навыки должны включать вспомогательные инструменты для разработки.

#### Scenario: Анализ изображения печатной формы
- GIVEN нужно определить границы колонок в макете
- WHEN агент использует `img-grid-analysis`
- THEN накладывается пронумерованная сетка на изображение

#### Scenario: Создание диаграмм
- GIVEN нужно визуализировать архитектуру
- WHEN агент использует `mermaid-diagrams`
- THEN получает шаблоны и правила создания Mermaid-диаграмм

### Requirement: Кросс-платформенная поддержка
Навыки должны учитывать различия Windows и Linux.

#### Scenario: PowerShell-команды на Windows
- GIVEN скрипт должен выполниться на Windows
- WHEN агент использует `powershell-windows`
- THEN получает правила корректного использования PowerShell: пути, Docker, HTTP
